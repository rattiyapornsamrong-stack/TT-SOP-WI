# TT-SOP-WI GitHub Pages

โครงสร้างนี้เตรียมไว้สำหรับอัปขึ้น GitHub Pages โดยตรง

## โครงสร้างไฟล์

- `index.html`  → หน้าเว็บ Viewer (มีรหัสผ่านกลาง)
- `docs/`       → โฟลเดอร์สำหรับเก็บเอกสาร PDF

ให้คุณนำไฟล์ของคุณมาวางในโฟลเดอร์ `docs` โดยใช้ชื่อ:
- `WI.pdf`  → Work Instruction
- `SOP.pdf` → Standard Operating Procedure

จากนั้นอัปโหลดไฟล์ทั้งหมดขึ้น GitHub repository แล้วตั้งค่า Pages:
- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`
